# NodeSchool Fort Worth

Welcome to the NodeSchool Fort Worth chapter's repository. NodeSchool Fort Worth is a monthly meetup to help people learn Node, JavaScript and other programming languages.

For more information on Fort Worth NodeSchool, please visit: <https://nodeschool.io/fort-worth/>

## Events

For information on upcoming events, checkout our Meetup page: <https://www.meetup.com/NodeSchool-Forth-Worth/>

## Questions

If you have any question please open a new new issue on github and we would be happy to help.

* Issues list (forum for question/answer): <https://github.com/nodeschool/fort-worth/issues/new>

## Contributing to the website

1. Fork the repository
1. Clone your fork to a local folder: `git clone <fork url here>`
1. `cd` into your project folder.
1. Update your changes
1. Push changes `git push origin master`
1. Navigate to [NodeSchool Fort Worth Github](https://github.com/nodeschool/fort-worth)
1. Submit pull request :smile:
